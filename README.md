# Packt Assessment

******** Setup step ********

- `composer install`
- create new .env file from .env.example and update database credential
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan db:seed`
- `npm install`
- `npm run dev`
- `php artisan serve`
